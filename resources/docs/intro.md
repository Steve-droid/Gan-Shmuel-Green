# Gan-Shmuel Project - Project Goals



## What You'll Learn This Week

This week is the most exciting and practical phase of the bootcamp. You'll work in cross-functional teams to build a complete production system, experiencing real-world software development challenges.

## Team Collaboration

Work as part of a development team using Git (feature branches, merges, pull requests)
Experience the need for DevOps from a developer's perspective
Experience development team demands from a DevOps perspective
Coordinate between dependent teams

## Technical Skills

Practice correct time management and task estimation (individual and team)
Initial exposure to microservice-based architecture
Familiarization with HTTP status codes in REST API responses
Build and consume APIs that other teams depend on

## Agile Practices

Use Kanban-style work boards (Trello) for task management
Practice daily stand-up meeting routines
Visualize and track work progress

## Important Constraint

Use only tools learned so far in bootcamp:

✅ Bash, Python, Flask, MySQL, Docker, Git


## The Real Learning
The biggest lessons come from:

Experiencing integration challenges between teams
Being a blocking dependency for another team
Understanding why automated testing matters
Learning that "it works on my machine" isn't enough


This week will be challenging, messy, and incredibly valuable!
